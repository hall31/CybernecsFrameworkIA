# 🚀 Démarrage Rapide - API Admin FastAPI

## ⚡ En 30 secondes !

### 1. Démarrer l'API
```bash
make start
```

### 2. Tester que ça fonctionne
```bash
make test
```

### 3. Ouvrir la documentation
```bash
make docs
```

## 🎯 URLs importantes
- **API**: http://localhost:8000
- **Swagger**: http://localhost:8000/docs
- **ReDoc**: http://localhost:8000/redoc

## 🔌 Prêt pour React !
Ton Dashboard React peut maintenant se connecter à cette API sur `http://localhost:8000` ! 🎉
